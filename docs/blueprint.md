# **App Name**: WasteWise

## Core Features:

- User Authentication: Secure login and signup page for all users.
- Waste Application Form: Department users can submit a waste application with details.
- Photo Upload: Upload photos of the waste generated.
- Address Autocomplete: Automatically suggest addresses based on user input.
- Geolocation Tagging: Embed location metadata (GPS) to photos, enabling the Waste Department to understand the application's precise location.
- Admin Dashboard: Admin (jpratap731@gmail.com) can manage and view all waste applications.
- Intelligent Routing Tool: Generative AI tool to optimize waste collection routes, improving overall process efficiency.

## Style Guidelines:

- Primary color: Soft Green (#A7D1AB) to represent sustainability and waste management.
- Background color: Light Gray (#F5F5F5) for a clean and professional look.
- Accent color: Light Blue (#B0E2FF) for interactive elements and highlights.
- Body font: 'Inter', sans-serif, for a modern, machined look that will make it very readable
- Headline font: 'Space Grotesk', sans-serif, giving a contemporary feel.
- Use simple and clear icons to represent different departments and waste types.
- Subtle animations to guide the user through the application process.
